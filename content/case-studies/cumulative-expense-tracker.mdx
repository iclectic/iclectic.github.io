## Overview

The Cumulative Expense Tracker is a Flutter based mobile application designed to help users monitor and visualise their spending patterns.

The project explores how simple mobile interfaces combined with data visualisation can help users understand financial behaviour more clearly.

## Problem

Many people track expenses across multiple tools or rely on manual spreadsheets.

This approach can make it difficult to observe cumulative spending patterns or detect financial trends over time.

## Solution

The application allows users to continuously add expense entries and visualise cumulative totals through dynamic charts.

The design emphasises clarity and ease of use so that users can quickly record expenses and see how their spending evolves.
