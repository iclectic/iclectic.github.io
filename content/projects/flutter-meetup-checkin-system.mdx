## Overview

A lightweight application designed to streamline attendee check ins at developer community events such as Flutter Birmingham meetups.

## Problem

Community events often rely on manual attendance tracking, which can be slow and difficult to manage as events grow.

## Solution

A digital check in system that allows organisers to quickly register attendees and maintain clean participation records.

## My role

I designed the system concept and implemented the application using Flutter.

## Stack

- Flutter
- Dart
- Mobile UI frameworks

## Key features

- QR based or simplified attendee check in flow
- Event attendance tracking
- Clean and lightweight user interface

## Challenge

Designing a simple check in process that would be fast for organisers during busy sessions.

## Outcome

Demonstrated how small developer tools can improve the organisation of community events and support ecosystem growth.
