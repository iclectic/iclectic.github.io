## Overview

A Flutter based expense tracking application designed to help users visualise cumulative financial activity.

## Problem

Many people track expenses manually or across fragmented tools.

## Solution

A mobile application where users can continuously add financial entries and visualise cumulative values over time.

## My role

I designed and developed the application using Flutter.

## Stack

- Flutter
- Dart
- Data visualisation libraries

## Key features

- Cumulative expense tracking
- Dynamic charts and visualisation
- Simple input interface

## Outcome

Explores how mobile apps can provide intuitive personal finance insights.
